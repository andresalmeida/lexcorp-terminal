## 🚀 Ejecución Local

### Prerrequisitos
- Node.js 18+ 
- npm o yarn

### Instalación

```bash
# Clonar repositorio
git clone https://github.com/andresalmeida/portfolio.git
cd portfolio

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev
```

El sitio estará disponible en `http://localhost:4321`
